Quick Start:

  • Click "Import Cookies" and select your cookie file (.json, .txt, or .csv)
  • Optionally specify a target domain for the imported cookies
  • Wait for import confirmation and check details if any cookies fail to import

Supported Formats:

  • JSON: Standard format with url and cookies array
  • Netscape: Traditional .txt format used by browsers
  • CSV: Simple format with domain, name, value columns
